# 项目文档

## 目录结构
主体目录结构
```
├─ api            存放接口 —— 以模块划分 
│  ├─ user     
│  └─ ...    
├─ assets         静态资源文件
├─ common         公共目录
│  ├─ axios       http
│  ├─ constant    常量
│  ├─ event       事件调度中心
│  ├─ socket      socket
│  └─ utils       工具类
├─ components     项目组件
│  ├─ module      业务组件
│  ├─ public      公共组件
├─ config         项目配置
├─ entities       实体类 —— 以模块划分 
│  ├─ user     
│  └─ ...  
├─ extension      vue扩展
│  ├─ directive   directive
│  ├─ router      router
│  └─ store       vuex
└─ views          页面组件 —— 以模块划分 
   ├─ user     
   └─ ...  
```
